# Python Mini Projects
A collection of Python scripts made by me.

## Projects
- YT-Downloader:

A script that uses `pytubefix, moviepy and tkinter` to search YouTube links, display them in a list of options wich the user is able to choose from the available streams to add the video to a download queue, also being able to choose the file type, (Video or Audio), and their qualities, ranging from video and audio quality and the order of download.
